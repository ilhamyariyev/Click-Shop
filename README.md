# What I use

* MVVM with Clean Architecture
* Material Design 3
* Coroutines & Flow
* Hilt for Dependency Injection
* Navigation Component
* Android JetPack
* LiveData
* Lifecycle
* ViewModel
* View Binding
* DataBinding
* Room Database
* Firebase Authentication
* Glide
* Retrofit
* Security Crypto
* Lottie
* ViewPager2
* CountDown View
